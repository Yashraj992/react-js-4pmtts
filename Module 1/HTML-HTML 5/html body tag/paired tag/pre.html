What is pre tag in HTML?

  • The pre tag defines performatted text.
  • Text in a pre element is displayed in a fixed width font, and the text preserves both spaces and line breaks.
  • The text will be displayed exactly as written in the HTML source code.
  • Example:
    • Lorem ipsum dolor, sit amet consectetur adipisicing elit. Nihil est aperiam quisquam, 
        quae nulla minus commodi amet mollitia ab ullam illo repellendus assumenda, corporis ea aliquam, esse fugiat minima architecto.
