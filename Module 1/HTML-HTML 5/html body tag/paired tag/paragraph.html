What is paragraph tag in HTML?

  • The p tag defines a paragraph.
  • These have both opening and closing tag.
  • Paragraph always starts on a new line, and browsers automatically add some margin before and after a paragraph.
  • Example:

    • Lorem ipsum dolor sit amet consectetur adipisicing elit. Perspiciatis quaerat ratione cum veritatis corporis, ipsam dolore facere corrupti nemo. Natus ea quidem minus illo autem maiores unde eveniet quibusdam in.
