What is audio tag in HTML?

  • With use of Audio tag, we can embed audio in webpage.
  • There are many attributes of <audio> tag.
  • autoplay:- When the page is loaded audio will be play as soon as possible.
  • controls:- It displays audio control.
  • loop:- Audio start again and again.
  • muted:- When the page is loaded audio will be automatically mute.
  • src:- URL of the audio file.
  • To embed audio in HTML, we can use the <audio> tag.
