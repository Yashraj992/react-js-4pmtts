What is HTMl meta tag?

  • The meta tag defines metadata about an HTML document.
  • Metadata is data about data.
  • Metadata will not be displayed on the page, but is machine parsable.
  • Metadata is used by browsers, search engines, and another end services.
    • > Meta data attributes:
            Content: Content is used to specify properties value.
            Author:  Name of author 
            Refresh: Automaticlly refresh website.
            Charset: This is used for indicating the character encoding for the HTML Document.
