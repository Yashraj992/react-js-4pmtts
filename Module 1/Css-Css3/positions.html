What is position property in CSS?

  • Position property is used to align the different elements in the HTML page.
  • Position property plays an important role to make high queality web pages.
  • Position property is used to define the position of the element on the web page.
  • By using the top, left, bottom, right, and z-index we can identify the exact of the element.
  • There are five position property in css:
  • Static (by default)
  • Relative
  • Absolute
  • Fixed
  • Sticky
