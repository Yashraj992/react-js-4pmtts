What is root-variables property in CSS?

  • CSS :root selector lets you define global variables that can be used in your stylesheets.
  • These variables, also known as custom css properties, providea convenient way of storing and reusing values, making your stylesheet more modular and easier to maintain.
  • To define a :root variable, use the :root selector followed by the variable name and value, surrounded by double dashes.
  • Example in style tag of head tag.
