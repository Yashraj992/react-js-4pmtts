What is flex-box property in CSS?

  • CSS flexbox, short for flexible box layout, offers a streamlined way to design adaptable and visually appealing layouts.
  • It works primarily in one row or column to intelligently distibute space among elements within a container.
  • Features of flexbox:
  • A lot of flexibility is given.
  • Arregement and alignment of items.
  • Proper spacing.
  • Order and sequencing of items.
  • Boostrap 4 is built on top of the flex layout.
