What is grid property in CSS?

  • It is a CSS property that offers a grid-based layout system, with rows and columns, making it easier to design web pages without floats and posotioning.
  • Property values:
  • Grid-template-rows/columns: It is used to soecifies the size of rows and columns.
  • Grid-template-areas: It is used to specifies the grid layout using named items.
