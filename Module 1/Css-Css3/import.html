What is import property in CSS?

  • The @import rule allows you to import a style sheet into another style sheet.
  • The @import rule must be at the top of the document.
  • The @import rule also supports media queries, so you can allow the import ot be media-dependent.
  • Property values:
  • url/string: A url or a string represents the location of the resource to be imported.The url may be relative or absolute.
  • list of mediaqueries: The list of media queries condition the application of the CSS rules defined in the linked url.
