What is selector property in CSS?

  • CSS selector are used to select the HTML elememts you want to style.
  • Types of css selectors:
  • (1)Simple selectors: Simple selectors select elements by it's name, id & class.
  • Class selectors:- Class selector apply with (.) and written class name.
  • Element selectors:- Element selector apply with name of element in style tag in head.
  • Id selector:- Id selector apply with (#) and written id name.
  • (2)Combinator selectors
  • Group selectors:- Group selector, write element in style tag by separated them with (,) symbol.
  • Child selectors:- Child selector used to apply to css child element of parent element.
  • General selectors:- General selector used to select the group of element that share the common parent element.
  • Adjacent selectors:- Adjacent selector is used to select an element that is directly after another specific element.
  • (3)Pseudo-class selector
  • Link selector:- Link selector is used to apply style on hyperlink.
  • Hover selector:- Hover selector is used to change style on element, When mouse over them.
  • Focus selector:- Focus selector is used to change style on element , When mouse over them.
  • (4)Pseudo-element selector
  • Placeholder selector:- Placeholder selector used to apply css to placeholder attribute. Placeholder is one type of attribute used in creation of login form.
  • After-Before selector:- After-Before selector in content property must be applied, without content it is dosen't work.
